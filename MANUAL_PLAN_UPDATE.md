# دليل تحديث خطة المستخدم يدوياً

بصفتك مالك المنصة، يمكنك ترقية حسابات المستخدمين من الباقة الأساسية (Default) إلى الباقات الأعلى (Pro / Business) عبر قاعدة البيانات مباشرة.

## 1. تحديث قاعدة البيانات (خطوة أولية - مرة واحدة فقط)

يجب عليك أولاً إضافة عمود `plan` إلى جدول الكتالوجات وتشغيل هذا الأمر في محرر SQL في Supabase:

```sql
ALTER TABLE catalogs ADD COLUMN IF NOT EXISTS plan TEXT DEFAULT 'basic';
```

## 2. كيفية تغيير خطة تاجر معين

لتغيير خطة مستخدم، تحتاج إلى معرفة `user_id` الخاص به أو `name` (اسم الكتالوج).

### الطريقة الأولى: عبر Supabase Table Editor (الأسهل)
1. افتح مشروعك في **Supabase Dashboard**.
2. انتقل إلى **Table Editor**.
3. اختر جدول **catalogs**.
4. ابحث عن السطر الخاص بالمتجر الذي تريد ترقيته.
5. في عمود **plan**، غيّر القيمة من `basic` إلى:
   - `pro` (للباقة الاحترافية)
   - `business` (لباقة الأعمال)
6. اضغط **Save** لحفظ التغييرات.

### الطريقة الثانية: عبر أوامر SQL
اكتب الأمر التالي في SQL Editor، مع استبدال `CATALOG_NAME` باسم الكتالوج المستهدف:

```sql
-- ترقية إلى Pro
UPDATE catalogs 
SET plan = 'pro' 
WHERE name = 'اسم_الكتالوج_هنا';
```

أو

```sql
-- ترقية إلى Business
UPDATE catalogs 
SET plan = 'business' 
WHERE name = 'اسم_الكتالوج_هنا';
```

## 3. التأكد من التغيير
بعد التحديث، عندما يقوم التاجر بالدخول إلى لوحة التحكم الخاصة به، سيظهر له شعار الباقة الجديدة بجانب عنوان "لوحة التحكم".

- **Basic**: باقة أساسية (أزرق)
- **Pro**: باقة احترافية (برتقالي/ذهبي)
- **Business**: باقة أعمال (بنفسجي فاخر)
